pwd - посмотреть где я нахожусь
clear - очистить терминал
find - поиск файла или папки
ls - показать содержимое папки ,в которой нахожусь
ls -la -показать все данные файлов(папок),в том числе и скрытые
mkdir - создать каталог
cd - перейти в каталог
touch - создать файл
cd . -в какой директории я нахожусь
cd .. -перейти в родительскую дерикторию
cat > " " - создаёт новый файл
cat "... " "..." > "... " -  объеденяет два файл и сохраняет содержимое в новый файл
mv "file"/ - перемещение файла 
mv "file" " " -  переминование файла
cp "katalog/file" "каталог/file - копирование файла и перемещение 
rm "katalog/file" - удаление файла
rm -r "каталог" -  удаление каталога 
cat >> "file" - добавляет в файла новые данные к старым
